# SubtitleTranslator

SubtitleTranslator is a python subtitle translator as a command-line tool.

### Installation

```sh
$ pip install https://github.com/Alireza-Gerami/SubtitleTranslator/archive/main.zip
```
### Usage

```sh
$ subtrans translate subtitle.srt --src en -dest fa
```
